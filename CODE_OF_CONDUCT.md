# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, and treat people the way
you'd want to be treated. Harassment, personal attacks, and bad-faith
participation aren't welcome.

If something happens that needs attention, report it to
**joseph.mcgrath@engineer.com**. Reports will be handled confidentially.
